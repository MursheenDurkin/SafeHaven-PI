# _teammate-originals

This folder preserves the **original HTML drafts contributed by team members** before they were integrated into the main project.

They are kept here for **academic integrity** — so anyone reviewing this repository can see exactly what each contributor produced in its unmodified form, separate from the integration work that followed.

## A note on TODO comments

These files contain `// TODO:` markers where backend integration was still pending at contribution time. Those TODOs are **resolved in the integrated production versions** that live in the project root (e.g. `login.html`, `portal-connected.html`, `business-admin.html`). The originals are intentionally left as-is here so the comparison is honest and the contribution credit clear.
